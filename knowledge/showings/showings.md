# Showings & Listings

The "Tenant First Impression Team"

As soon as we build a listing and publish it on our website, it automatically gets listed on our partner syndications. People showing interest there can contact us and their queries land in the "Manager: Listings" inbox. When any contact books a showing, they get sent information through a number tied to our "Manager: Showings" inbox.

## Prospect Showing Flowchart - Potential Blockers & Solutions

### Phone Check
- Issue: Using a VoIP based phone number (like Google Voice)
- Solution: Require actual phone number

### Anonymous IP
- Issue: Likely using a VPN
- Solution: Encourage them to turn off VPN; assure privacy compliance

### Distance Check
- Issue: IP traced far from property
- Solution: Present Stripe ID verification option or wait until they're nearer

### Distance vs Time
- System estimates if prospect can make showing slot at 100km/h average speed
- If not possible today: No Stripe bypass, must book another day or change IP
- If tomorrow or later: Stripe bypass available

### Risk Score
- Maxmind identified IP or IP + Email/Address as high fraud potential
- Solution: Use different connection and potentially different email; consult manager

### Previously Flagged IP/Phone
- Automatically offered Stripe verification
- If successful, sent access code

### Blocked Status
- Prospect observed in suspicious activity or confirmed scammer
- Never mention they're blocked; consult manager on course of action

## Stripe ID Verification

## General Information & Policies

### Listings vs Showings Inbox

**Listings Inbox receives:**
- SMS to property's dedicated number
- Calls to property's dedicated number
- Emails to property's dedicated email or listings@ziprent.com
- Queries from syndication partners (Zillow, Rentals.com, etc.)

**Showings Inbox receives:**
- Responses from booking confirmation SMS
- SMS/email from prospects who booked a showing

### Private Showing vs Open House

**Private Showing:**
- Prospect books for vacant property
- Gets lockbox code for self-guided tour
- Most properties are vacant
- Available in 2-hour window

**Open House:**
- For occupied properties or rare owner-occupied
- Fixed 1-hour slots (usually 1 per day)
- Maximum 6 prospects per showing
- Occupants get email notification; system stops booking at 6

### Applications

Prospects can apply for one property at a time. If they don't secure it, applications can be moved to another available property using 'Move' under 'Actions' dropdown.

### Rescheduling a Showing

Prospects get unique cancellation link in SMS. They can use it to cancel existing showing and book another slot. System doesn't allow multiple bookings for single property at same time.

### Attempts to Negotiate

Anyone wanting to negotiate rent price should be encouraged to fill out an application first. With completed application, owners can make informed decisions.

### Corporate Lease

Ziprent does not work with Corporate leasing (situations where tenant changes during lease term).

### Insurance Companies

Usually short-term leases which we don't do. If for TP Only, can present to owner. If property not performing well, may fill house temporarily. Full year requires normal application process with insurance information attached.

### Lease Break Protocol

If tenants must vacate, they should cooperate with showings. Ziprent will locate, vet, and sign new lease at current placement cost (paid by tenant). Tenants pay all rent until new tenant's move-in date. Minimum 5-day gap required between departures for potential work.

### Lease Terms

All Leases are for 12 months unless otherwise stated in the Listing.

### Third Party Payment

Rarely accepted. Must be named person added to lease paying via ACH. Checks not accepted.

## Applicant FAQ

### Foreign Applicants
- General criteria: 3x rent in combined income, 650+ credit score
- No established credit: require cosigner
- One applicant meets requirements: may ask for higher deposit

**Note:** Starting 7/1/2024, CA Assembly Bill 12 limits security deposits to 1 month's rent for small landlords (natural persons or LLCs owning max 2 properties with max 4 units).

### No Paystubs
- Connect bank account through Payscore on application form

### Home Daycare
- Per California law, cannot deny daycare operators
- Ask them to include business information in application for owner's review

### Cancellation Response
"Thank you for letting us know. Are you interested in seeing it in the future? Should you wish to view our other available properties: https://www.ziprent.com/available-listings"

### Speaking to Owner
- Ziprent serves as property manager intermediary
- No direct contact with landlord unless per their request

### Later Move-In Date
- Properties move quickly, owners don't want to lose rent
- Encourage viewing properties closer to desired move-in date

### Can't Get in Lockbox
- Clarify: unable to open lockbox or unable to open house?
- Offer to schedule another showing

### Rental Criteria
- 3x rent in combined income across all applicants
- 650+ credit score for all adult occupants
- May accept higher deposit or cosigner instead of requirements

### Distance Error
- Sign up for tour when closer to property
- Sign onto WiFi for proper IP detection
- Restart phone if recently traveled

### Section 8
- We work with Section 8
- Applicants must include voucher with application
- Provide proof of income for portion not covered by voucher
- Section 8 template available in Front

## Lockbox Troubleshooting

### Quick Diagnostic
- Is prospect at right time/date?
- Is SLB lighting up and beeping?
- If not lighting up: have them gently hit face to close completely
- Is alarm going off? (code entered wrong 3 times = 10 minute reset)
- Does box number match Ziprent? (2-3 digit number on back)
- Any repeating numbers in code? Check beep for each press
- Keypad not responding? Wipe front with shirt, press back button, try again
- Near end of time? Generate longer code in Ziprent

### Common Issues

**Lockbox Not There:**
- Tenant at wrong door
- Check lockbox instructions in system
- Search email in Front for what instructions they received

**Code Won't Work:**
- Wrong lockbox (we have classic and smart at same property)
- Match # on back of lockbox with system
- Rare: lockbox died/malfunctioned, swapped lockbox
- Check default lockbox code on property page

**Wrong Time:**
- Check their showing time under Showings
- If already registered and no conflicts: give new code
- Ensure they don't hand key to anyone; return key to lockbox

**Not Making Beep for Each Number:**
- Must listen for beep after each number
- Must press center button after code

**Got Code Right, Don't Know How to Open:**
- Watch YouTube video
- Press center lock icon
- Pull on notches on either side
- Entire face pulls down

**Constant Beep or Alarm:**
- Usually from wrong code entered too many times
- Lockbox unresponsive 5 minutes after 3 wrong tries

**Lockbox Malfunction:**
- Lockbox dead: doesn't respond or numbers turn red
- Lockbox missing: could be stolen
- Lockbox jammed: rarely happens; unlock heard but won't open
- Encourage gentle force (hit down on front with bottom of fist while holding back)

### If Can't Get Them Inside
1. Respond to email to track issue
2. Search correspondence in Front
3. Call/text last few showings
4. If isolated: monitor and follow up
5. If happening to others: notify "All of Ziprent"
6. Tag onboarding staff in thread
7. Look up original inspector to fix
8. Notify future showings once issue determined
9. Notify owner when appropriate

### Code Generation Details
- Codes work offline via internal algorithm
- Code valid for 24 hours from "Start Time/Date"
- Example: Code starts Oct 1 at 10am, valid until Oct 10 at 10am
- Tenant must use within first 24 hours or code expires
- If not used in first 24 hours, new code needed starting from next use time
- Codes cannot be changed or extended
- For extended use (vendors, Airbnb, smart deadbolt): inform guests code can't be easily changed once given out
