# KeyCafe

KeyCafe Instructions

## Overview

For properties in San Francisco and Oakland where lockboxes are not allowed we can utilize a service called KeyCafe to handle key exchanges, access for inspectors/vendors, and for tenant access upon move-in. Below are the various situations and configurations to help make it a smooth transition.

Please note the KeyCafe app is not the best and can be a bit frustrating to use. If you or a vendor experience issues with Key Cafe, it is best to reach out to their customer service via chat for the quickest resolution.

## How to Add a Key for a Property (That Has Not Been Set-Up Previously)

### Before Adding a Key

Use "Search keys" area to check if a key area exists in the system. There should never be a duplicate key made for an existing property.

### To Add a New Key

1. Click on +Add at the top of the page
2. Click "KEY" in the pop-up
3. Type in the address name as the KEY NAME and click "ADD KEY" at the bottom
4. Make sure to include the UNIT number as we may have multiple units within the same building
5. Click "OKAY" on the confirmation pop-up

### Setting Up the FOB

1. On the page that appears, locate "Set Up Key with FOB"
2. This is where the serial number for the FOB that the photographer/inspector will need to confirm will be placed
3. Next, go to the box where it says "Set up KeyCafe Public Network"
4. Click "FIND ON OUR MAP"

### Finding the KeyCafe Location

1. A map of all KeyCafes will pop up
2. Search the Address where it is written "San Francisco, CA, USA"
3. Click on the white bubble with the orange and blue rings for your desired location
4. You will see the location name, and a blue arrow to get more details
5. Click the blue arrow to see the photo of the Kiosk and the Address
6. Note if bins are available
7. Send this address to the inspector with the location name

## Setting Up a NEW Key with a KEY CAFE FOB

Any new keys dropped off at a Key Cafe need to have a Key Cafe FOB attached to track the keys within the machine.

If an inspector/photographer is dropping off keys that need to be attached to a KEY CAFE FOB, they will need to use this **ACCESS CODE to dispense a Key Cafe FOB:** `14156886660`

After using the access code to get the FOB, they will use their one-time or ongoing booking code to drop off the keys.

## Adding a One-Time or Ongoing Booking Code

For an owner or inspector/photographer who is hosting showings on a regular basis, you can use "Add Ongoing Booking":

1. Type in the visitor's name and email
2. Click ADD
3. This will provide a one-time, permanent code for that person to use at the KeyCafe for all needed visits
4. A pop-up will appear - click "copy details" to copy and paste them in an email to the inspector with specific instructions
5. Click OKAY after you have copied the details

**NOTE:** "Ongoing Booking" codes should NOT be sent to vendors/inspectors with a single assignment/job/appointment. Those will need to be issued a "One-Time Booking" code.

### Email to Inspector Placing the Keys

**IMPORTANT:** The email to the inspector must include the following:

1. Address of KeyCafe location where key will be placed
2. A reminder that we need to be sent the serial number after they place the keys
3. The copied details with their booking code and the key name
4. Outline that the fob MUST be attached to the keys

The inspector will:

1. Go to the KeyCafe Location once they have the keys
2. Select the key name and use their assigned booking code to drop them off
3. Be given a fob from the machine to attach to the keys
4. Send you the serial number of the fob once they have placed the keys

After receiving the serial number:

1. Go back to the key's main page
2. Click "Set up key fob"
3. Type in the serial number and click LINK

## How to Send Booking Codes to Vendors/Owners/Tenants

### Finding the Property

1. Go to the top of the page next to the +Add link
2. Type in the property name
3. Select the property below once you find it by clicking on it on the left

### Sending Booking Codes

Send an access code to the owners with either "Add One-Time Booking" or "Add Ongoing Booking":

- For the owner or an inspector/photographer hosting showings use "Add Ongoing Booking"
- For everyone else use "Add One-Time Booking" with a timed code based on when they will collect the keys

### Add One Time Booking

1. Type in the name and email/phone number for whoever you are making the code
2. Click Add
3. Click copy details and it will automatically copy the codes and pick up information
4. Email this information to whoever is using the keys
5. KeyCafe will also automatically send them the booking code and location information

### Add Ongoing Booking

1. Type in the visitor's name and email and click Add
2. Click copy details and it will automatically copy the codes and pick up information
3. Email this information to the owner
4. Make sure to send the correct address information for where the KeyCafe is located (They are normally in convenience stores)
5. Click "Home Location" to find this information

### Viewing Bookings

1. Click "Bookings" to see who has been given access and if their access is One-Time or Ongoing
2. Click on the name with the code you want
3. You will be able to copy details and send updated information if needed

## Add KeyCafe Information as a Lockbox

Once the key is set-up and has been placed in the Key Cafe, we will add a lockbox labeled as KEYCAFE to the LOCKBOX page in the dash.

**CODE:** Input - "To be sent/set-up as needed via Key Cafe"

*NOTE: Universal ongoing booking code is NOT to be added to the lockbox. Each vendor needs individual access provided.*

**LOCATION:** Input Address and location of Key Cafe where keys have been placed

**INSTRUCTIONS:** Add Key Name and Pick-up/Drop Off Instructions

### PICK-UP Instructions

1. Once you are at the correct Keycafe SmartBox location, locate the Keycafe SmartBox
2. Touch the SmartBox's screen to begin
3. Select Key Pickup
4. Enter your access code and press Enter
5. For additional security, you may receive a 4-digit SMS code to your mobile number that you will need to enter into the SmartBox
6. Select the name of the keys that you wish to pick up and press Pick Up
7. The designated key bin in the SmartBox will open to give you access to the keys
8. Once you have picked up the keys, close the key bin door firmly

### DROP-OFF Instructions

1. Locate your Keycafe SmartBox
2. On the SmartBox touch screen, Touch Screen to Start
3. Select Key Drop Off
4. Enter your Access Code (this is the code associated with your Keycafe account or the code you received in your access communication)
5. Select Enter
6. Scan the key fob by placing the fob that is attached to your key on the scan area of the SmartBox (usually on the right side of the box or right of the screen)
7. Upon scanning, you will hear a beep sound followed by a key bin door opening
8. Place your keys in the open key bin and close the key bin door firmly

## How to Remove a Key from Key Cafe

Once a new tenant moves in or we are no longer managing a property, we will want to fully remove the keys from KeyCafe.

Follow the instructions above on how to send booking codes or send them the ongoing access from our Lockbox page.

**IMPORTANT:** Tenants/Owners must remove the fob from the keys and return it to the key cafe when they pick up the keys after tenant move-in.

### Retiring a Key

Once it is confirmed that the tenant/owner has removed all keys from KeyCafe AND removed the fob:

1. Go to "About Key" and update the Key Name and add "-LEASED"
2. Remove the FOB# (serial number)
3. Click edit to do this and then save once finished
4. Do this when it is leased and it may be used again in the future
5. Make sure it says "Key Checked Out" as well

If you want to fully remove the property and key from the KeyCafe:

1. Go to "About Key"
2. Click "Retire Key" at the bottom
3. This will remove this property fully from the system

## Additional Notes on Providing Access / Pick Up

Once a key is placed inside of the KeyCafe, the details of the Keycafe location and Key name should be added under the 'Lockbox' tab of our Ziprent portal. Individual "One-Time Booking" codes for vendor or tenant use are to be issued via the Key Cafe site, as-needed.

### In-Person Showings

If we have to store the keys there while we are doing in person AGENT HOSTED showings, we can use the "Add Ongoing Booking" code. The person picking up and dropping the keys off will just be using the same code provided to them.

**KeyCafe will never be used for Self-Guided Showings and should never be provided to prospects for access.**

### Key Pick Up

Should new tenants be picking up the keys from the Keycafe:

1. Use the "Add One Time Booking" function to generate a code for the new tenants on their pick up date
2. In your instructions to the tenants for their pick-up, make sure you have them remove the KeyCafe fob from the key set
3. Make sure they confirm they've done that and then we can remove the key from our account
