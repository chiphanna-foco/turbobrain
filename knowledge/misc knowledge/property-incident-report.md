# Property Incident Report

How to create and store property incident report

## Filing a Property Incident Report

Below are instructions on how to file an accident report for properties that don't have tenants in them. This is to make sure all relevant information, for break-ins, re-keys, vandalization, etc… are stored in an easy to read place.

**Form Link:** https://forms.gle/wgi4yw71vb1XohCbA

### What Needs to Be Filled Out

All sections need to be completed. Below is what needs to be included:

- **Address**: Include the full address, Street, unit number, and City
- **Date**: Put the date we received the notification (additional dates can be included in the "What Occurred" box)
- **Incident Details**: Include names, dates, any other information that is important
- **Items Left in House**: IF ITEMS WERE LEFT IN THE HOUSE THEY NEED TO BE COMPLETELY DOCUMENTED AND PHOTOS TAKEN OF THE EXACT ITEMS
- **Scams**: Make sure if it was a scam we get the screenshots of where the scam was, along with the numbers and email addresses used (screenshots can be uploaded at the end)
- **Resolution**: Include what has been done to resolve this. If it is ongoing, include everything that has happened so far (no additional form needs to be submitted)
- **Thread Link**: Include ONE (the main) front link to the issue - all things should be cross referenced in that thread
- **Photos**: Upload photos of the issue that occurred; document as much photo proof as possible
- **Submit**: Click the submit button and the document will be emailed to you

## After Submission Process

Once you've submitted the form:

1. Go to your personal inbox and move the document to the Manager Inbox
2. Change the title of the email in the thread to include the address of the property
3. Crosslink the primary thread as a comment on the moved Google Form and pin it to the email
4. Go to the property page and add a comment detailing what happened and linking the Front thread in the body
5. At this point you are done with storing the information - this will allow us to easily reference issues
